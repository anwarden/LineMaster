# Vodoo Game

A fast-paced action game built with TypeScript and Vite.

Master the lines, defeat your enemies, and complete challenging levels in this arcade-style adventure. Features strategic gameplay, multiple stations, and exciting visual effects.

## Play Online

🎮 [Play on itch.io](https://nosleepnoe.itch.io/line-master)

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
